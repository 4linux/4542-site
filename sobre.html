Sobre o OKD

O OKD é a versão free
e open source do Openshift

Seu propósito é facilitar o ciclo de vida das aplicações, facilitando a interação entre os programadores e o cluster. O principal elemento responsável por essa facilidade é o Web Console.

Docker

Docker é a tecnologia por trás do isolamento das aplicações provisionadas dentro do Openshift, possibilitando a criação de novas imagens e facilitando todo o processo de desenvolvimento.

Kubernetes

Kubernetes é o motor do Openshift, o orquestrador responsável pelo ciclo de vida dos contêineres. Foi adicionado na versão 3 do Openshift, substituindo o motor original.

Openshift

Openshift é uma plataforma de PaaS que utiliza contêineres para isolamento de aplicações orquestradas através do Kuberentes. Seu principal objetivo é agilizar e facilitar o fluxo de desenvolvimento e manutenção das aplicações entre diversas equipes.
